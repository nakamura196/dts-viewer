# About DTS Viewer

DTS Viewer is a web application for exploring and searching text collections through the DTS (Distributed Text Services) API standard.

## What is DTS?

DTS (Distributed Text Services) is a specification for exposing collections of texts and their metadata in a RESTful API. This standard enables interoperability between digital text repositories and applications.

## Features

- Browse text collections hierarchically
- View detailed metadata using Dublin Core standards
- Download texts in various formats
- Navigate citation structures
- Multi-language support (English/Japanese)

## Getting Started

Enter a DTS API endpoint URL in the form on the home page to start exploring text collections.
