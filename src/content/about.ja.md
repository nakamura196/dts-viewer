# DTS Viewer について

DTS Viewer は、DTS（Distributed Text Services）API 標準を通じてテキストコレクションを探索・検索するための Web アプリケーションです。

## DTS とは？

DTS（Distributed Text Services）は、テキストコレクションとそのメタデータを RESTful API で公開するための仕様です。この標準により、デジタルテキストリポジトリとアプリケーション間の相互運用性が実現されます。

## 機能

- テキストコレクションの階層的な閲覧
- Dublin Core 標準を使用した詳細なメタデータ表示
- さまざまな形式でのテキストダウンロード
- 引用構造のナビゲーション
- 多言語サポート（英語・日本語）

## はじめに

ホームページのフォームに DTS API エンドポイントの URL を入力して、テキストコレクションの探索を始めてください。
